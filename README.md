# Sogverse

Sogverse connects parents with game educators (GEDUs) for guided gaming sessions. Parents purchase sessions and manage their children's accounts, while GEDUs lead groups through curated gaming experiences with built-in voice chat.

## User Roles

- **Customers** — Parents who purchase products, manage linked gamer accounts, and enroll them in groups
- **Gamers** — Children with simplified username-based accounts
- **GEDUs** — Game educators who run group sessions with voice/video chat
- **Admins** — System management and oversight
